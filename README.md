# regression-model
This project demonstrates the implementation of Simple Linear Regression using Python and the Scikit-learn library. The objective is to predict house prices based on their area by learning the linear relationship between the input feature (house area) and the target variable (house price). The project begins by importing the required libraries and loading the dataset, followed by visualizing the data to understand its distribution. A Linear Regression model is then trained to learn the best-fit line, which is used to make predictions for new house areas. The project also examines the model's coefficient (slope) and intercept, verifies the regression equation, predicts prices for multiple houses, and saves the predicted results into a new CSV file. This project provides a clear understanding of how Simple Linear Regression is applied to solve real-world prediction problems.
